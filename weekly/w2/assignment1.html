Well Designed Websites

princess polly homepage

I believe this is a well-designed website because when you first arrive at the page since it is an Australlian website it can tell where you access this website from and it allows you to choose if if you want to visit the USA store or the AUS store. It also has tabs to surf the website and allows the website to be easily accessible. It also allows you to shop through their Instagram. It also has nice graphics which is a nice color tone. There is nothing that I do not like about this website other than when clicking on a photo, it takes you all the way back up to the top.

amazon homepage

Amazon has a nice layout it organizes the website to allow you to see your past searches and items you might be interested in. It also has many categories and subcategories. It shows deals that are under $25. It also has boxes that allow you to see the images better and also allows it to pop more. The only thing that I do not enjoy about this website is that it can be overwhelming to look at the products.
